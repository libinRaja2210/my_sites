Multi operational Datatable In PHP (Footer Callback, Export Reports, Hide Column, Date Range Filter, Individuval Search).

WebSolutionTricks
PHP Datatable With Footer Callback,Date Range Filter,Individual Search,Column Hide And Unhide Option,Export Reports options.

Multi Optional Datatable in PHP:

Every day we are see so many Datatable available in the internet.But many of datatable has limited feautres only available.Commonly datatable using read the database value.Datatable has two type of processing Clint-side Processing & Server-side Processing.This datatable has both processing are possible and real time Datatable.The other features are Footer Callback, Export Reports, Hide Column, Date Range Filter, Individuval Search working fine this datatable.

How To Create Multi Optional Datatable System Step by Step Guide:
  1. Create Database.
  2. Insert database Values.
  3. Create index page
How Its Works Demo Video Click Below:

CREATE TABLE IF NOT EXISTS `datatable` (

			var api = this.api();
				nb_cols = api.columns().nodes().length;
				var j = 3;
				while(j < nb_cols){
					var pageTotal = api
                .column( j, { page: 'current'} )
                .data()
                .reduce( function (a, b) {
                    return Number(a) + Number(b);
                }, 0 );

          $( api.column( j ).footer() ).html('₹'+pageTotal);
					j++;

// Extend dataTables search
$.fn.dataTable.ext.search.push(
  function(settings, data, dataIndex) {
    var min = $('#min-date').val();
    var max = $('#max-date').val();
    var createdAt = data[2] || 0; // Our date column in the table

    if (
      (min == "" || max == "") ||
      (moment(createdAt).isSameOrAfter(min) && moment(createdAt).isSameOrBefore(max))
    ) {
      return true;
    }
    return false;
  }
);
